# Plixom Website

## วิธีเปิดเว็บไซต์

เพียงแค่เปิดไฟล์ `index.html` ในเบราว์เซอร์

หรือใช้ Live Server:
```bash
# หากมี Python ติดตั้งอยู่
cd /Users/Aum/development/plixom/web
python3 -m http.server 8000
```

จากนั้นเปิด http://localhost:8000 ในเบราว์เซอร์

## ไฟล์ที่สร้าง

- `index.html` - หน้าเว็บหลัก
- `styles.css` - สไตล์ที่สวยงาม พร้อม animations
- `script.js` - JavaScript สำหรับ interactive features
- `logo.png` - วางโลโก้ของคุณที่นี่
